# DixX Save

**Скачивание видео и аудио с YouTube, TikTok, Instagram, VK и многих других сайтов — прямо на устройстве.**

</div>

## Возможности

- 🎬 Скачивание **видео** в выбранном качестве
- 🎵 Извлечение **аудио** (в том числе MP3)
- 🌐 Поддержка множества платформ
- ⭐ **Premium-подписка прямо в приложении** (оплата внутри); по желанию — привязка Telegram-аккаунта, чтобы объединить подписки приложения и Telegram в одну
- 🔄 Встроенная проверка и установка обновлений

## Установка

1. Откройте раздел **[Releases](../../releases)** и скачайте APK последней версии.
2. Выберите файл под свой телефон:
   - **`arm64-v8a`** — большинство современных телефонов (рекомендуется);
   - `armeabi-v7a` — старые устройства;
   - `x86` / `x86_64` — эмуляторы / отдельные устройства;
   - **`universal`** — если не уверены (работает везде, но больше по размеру).
3. Разрешите установку из неизвестных источников и установите APK.

> ℹ️ При установке Android может показать предупреждение **Google Play Protect** — нажмите «Установить без проверки». Это нормально для приложений, устанавливаемых не из Google Play, и не связано с безопасностью самого приложения.

## Обновления

Приложение само проверяет наличие новой версии и предлагает скачать и установить обновление. Также новые версии всегда можно скачать вручную из раздела **Releases**.
