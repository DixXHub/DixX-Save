## 📱 Приложение для Android

Самый быстрый и удобный способ. Приложение скачивает файлы **напрямую на устройство — без ограничений по размеру и длительности.**

**Возможности:**

- 🎬 Видео в любом качестве — от 360p до 4K, по вашему выбору
- 🎵 Аудио и **MP3** с обложкой и метаданными
- 📋 Скачивание целых **плейлистов** одной кнопкой
- 📝 Встраивание **субтитров** в видео
- ⚡ Ускоренная загрузка через встроенный **aria2c**
- 🗂 История загрузок и шаблоны команд
- 🎨 Современный интерфейс **Material You** в фирменной оранжево-чёрной теме
- 🔄 Встроенная проверка и установка обновлений

### Установка

1. Откройте раздел **[Releases](../../releases/latest)** и скачайте APK последней версии.
2. Выберите файл под свой телефон:
   - **`arm64-v8a`** — большинство современных телефонов *(рекомендуется)*;
   - `armeabi-v7a` — старые устройства;
   - `x86` / `x86_64` — эмуляторы и отдельные устройства;
   - **`universal`** — если не уверены *(работает везде, но весит больше)*.
3. Разрешите установку из неизвестных источников и установите APK.

> ℹ️ Android может показать предупреждение **Play Protect** — нажмите «Установить без проверки».
> Это нормально для приложений не из Google Play и не связано с безопасностью DixX Save.
